Bienvenido a mi portafolio

Este portafolio está enfocado en mis proyectos de diseño y desarrollo web, donde combino la creatividad con la funcionalidad para construir páginas modernas, atractivas y fáciles de usar. Aquí podrás encontrar ejemplos de maquetación, uso de HTML, CSS y técnicas de diseño responsivo que permiten que cada sitio se adapte de manera correcta a diferentes dispositivos y pantallas. Mi intención es mostrar cómo una buena estructura visual y un código limpio pueden trabajar juntos para ofrecer experiencias digitales claras, intuitivas y profesionales.
